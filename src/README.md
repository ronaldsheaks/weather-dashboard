# Weather Dashboard Phase I

A responsive weather dashboard built with React and Vite that lets users search for a location, view it on an interactive map, and see current weather details plus a short forecast.

## Features

- Search by city, state, or place name
- Interactive map with a cleaner CARTO basemap
- Centered weather/location display
- Current weather widgets
- Short forecast section
- Loading states for search and weather fetches
- Cleaner no-result and error messaging
- Improved spacing and UI polish

## Tech Stack

- React
- Vite
- Leaflet
- CSS
- Weather API
- Geocoding / location search API
